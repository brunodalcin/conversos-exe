# **CONVERSOS**

**Conversos** was created only with study purposes, its a lightweight, offline image converter and compressor built with **FreePascal 3.2.2**, **IDE Lazarus 4.2** and the **FPImage** framework. 
It allows private conversion between common image formats and re-encoding with adjustable compression without relying on online tools or ad-heavy websites.

---

## **Features** 
- Converter Supports **JPG**, **PNG**, **BMP**
- Compressor Supports **JPG**, **PNG**
  - Set desired compression level.
- Fully **offline**, no internet access needed.
- Simple and clean UI.
